Data Types 5 major Data types

1. String;A sequence of 16-bit unsigned integer values encoding UTF-16 code units is used to represent textual data as a String type.

2. Number; Number is a representation of decimal and floating-point numbers, including exponentials.

3. Bigint; The JavaScript BigInt type is a primitive numeric type that may represent integers of any size. Even after the safe integer limit, you can store and work with huge integers safely with BigInts.

4. Boolean; A boolean data type gives true or false as its only two possible values. In JavaScript, the Boolean function is used to determine whether a variable, object, condition, expression, etc. has a true or false value.

5. Undefined; this refers to a variable that has been declared but not yet given a value. Undefined stands alone as a type.
